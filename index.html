🌿 Modular Game Engine

Personal reference for building 1-page modular games and tools.

📁 Directory Structure

modular-game-engine/
├── core.js
├── input.js
├── save.js
├── ui.js
└── style.css

🧩 How to Use in a New Project

<link rel="stylesheet" href="https://yourname.github.io/modular-game-engine/style.css">
<script src="https://yourname.github.io/modular-game-engine/core.js"></script>
<script src="https://yourname.github.io/modular-game-engine/input.js"></script>

✅ Do & ❌ Don’t

✅ Do❌ Don’t
Use modules only when neededLink unused scripts
Keep it to one HTML pageOverload with features
Use canvas and divs for layoutAssume server features

🧠 Notes to Self

  • You can copy modules instead of linking them
  • Comment logic and keep code clean
  • Modules are meant to be optional
